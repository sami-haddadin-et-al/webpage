# Mobile FR3 Duo (Prototype)


Mobile FR3 Duo is an all-in-one robotic system that combines Tactile Mobile Robot (TMR) with FR3 Duo. It integrates mobility, perception, and dual-arm manipulation, making it ideal for data collection and real-world AI deployment.
